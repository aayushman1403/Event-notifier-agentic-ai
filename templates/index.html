Setup Instructions

To use this notifier, you need to set up the following Replit Secrets:

  • SENDER_EMAIL: Your Gmail address (e.g., yourname@gmail.com)
  • SENDER_EMAIL_PASSWORD: Your Gmail App Password (not your regular password)
  • RECEIVER_EMAIL: Email address to receive notifications
  • DEFAULT_URL: (Optional) Default URL to monitor
  • DEFAULT_KEYWORDS: (Optional) Default keywords, comma-separated
  • CHECK_TOKEN: (Required for API) Security token to protect the /check API endpoint from abuse

Important: Email credentials are only read from Replit Secrets for security. They are never stored in files.

How to get Gmail App Password:

  1. Go to your Google Account settings
  2. Enable 2-Step Verification if not already enabled
  3. Go to "App passwords" section
  4. Generate a new app password for "Mail"
  5. Use this 16-character password as SENDER_EMAIL_PASSWORD

Automated Monitoring

For automated checks, set up an external service (like UptimeRobot) to ping the /check endpoint at regular intervals.

If you set a CHECK_TOKEN secret, include it as either:

  • Header: X-Check-Token: your-token
  • Query parameter: /check?token=your-token
